About This Project

AI-Based Job Recommendation System using Flask, Machine Learning, NLP, and Resume Parsing.

Project Overview

This project is an AI-powered web application that analyzes uploaded resumes, extracts skills using Natural Language Processing (NLP), and recommends suitable jobs based on skill matching.

The system also identifies missing skills and recommends online courses to improve career opportunities.

Key Features

Resume Upload

Upload PDF resumes for AI analysis.

Skill Extraction

Automatically extracts technical skills.

Job Recommendation

Suggests jobs based on resume skills.

Course Recommendation

Recommends courses for missing skills.

Technologies Used

Python Flask Machine Learning NLP Bootstrap 5 HTML CSS SQLite

AI Recommendation Logic

The recommendation engine analyzes resume content, extracts important technical skills, compares them with job requirements, and calculates a match score.

Based on missing skills, the system also suggests learning resources and courses to improve employability.
